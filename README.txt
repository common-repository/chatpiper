== Description ==
This is just the extension to enable the app, NOT the app itself.
For details on the app/buying the app: https://www.chatpiper.com/

ChatPiper is a chat system that is installed on your own server.

== Installation ==
1. From https://chatpiper.com download and install on your own server ChatPiper
2. Upload `plugin-name.zip` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. In wordpress at the ChatPiper's section enter the URL to the aplication on your server.
